Question Option

A few of the Question Types have an option which is activated by clicking on the checkbox. The question types and the meaning of the options are detailed below.

  1. Multiple Choice There is variant of Multiple Choice questions called "Multiple Choice Multianswer" questions. If the Multianswer option is selected then the student is required to select all the correct answers from the set of answers. The question may or may not tell the student how many correct answers there are. For example "Which of the following were US Presidents?" does not, while "Select the two US presidents from the following list." does. The actual number of correct answers can be from one up to the number of choices. (A Multiple Choice Multianswer question with one correct answer is different from a Multiple Choice question as the former allows the student the possibility of choosing more than one answer while the latter does not.)

  2. Short Answer or Cloze You can make the answers to the Short Answer or Cloze questions case sensitive by checking the Case Sensitive checkbox. If you do not check this box, the answer will not be case sensitive. For example, if the answer to a question is Timbuktu, checking Case sensitive will require the uppercase T at the beginning of the word. Without case sensitivity, either Timbuktu or timbuktu will be accepted.

The other Question Types do not use the Question Option.
