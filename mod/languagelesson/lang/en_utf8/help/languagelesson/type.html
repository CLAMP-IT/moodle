Lesson Type

Practice

A Practice lesson will NOT appear as an entry in the Gradebook for this assignment. It also defaults to showing default feedback and disables the option to show ongoing score (because scoring is irrelevant).

Assignment

An Assignment lesson will appear as an entry in the gradebook and allow access to customize all feedback and grading options.

Test

A Test lesson will appear as an entry in the gradebook, and sets the default number of (question) attempts to 1, and also sets the Display Previous Answer to No. However, these options can be changed.
